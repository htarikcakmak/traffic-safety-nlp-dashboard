## 📌 Overview
This project is an interactive data analysis and Natural Language Processing (NLP) dashboard built with **Python** and **Streamlit**. It is designed to analyze and visualize traffic safety data, providing actionable insights through text processing and data visualization. 

This application demonstrates my ability to build data-driven web applications, handle datasets, and deploy interactive user interfaces using modern Python frameworks.

## ✨ Key Features
* **Interactive Dashboard:** Built a responsive and user-friendly interface using Streamlit.
* **Data Processing & Analysis:** Processed raw traffic datasets to extract meaningful safety metrics.
* **NLP Integration:** Implemented Natural Language Processing to analyze text-based traffic reports/sentiments.
* **Data Visualization:** Generated clear and readable charts for trend analysis.

## 🛠️ Tech Stack
* **Language:** Python
* **Framework:** Streamlit
* **Data Analysis:** Pandas, NumPy
* **NLP & Visualization:** Hugging Face Transformers, Matplotlib, clean-text

## ⚙️ How to Run Locally
If you want to run this project on your local machine, follow these steps:

1. Clone the repository:
   git clone https://github.com/your-username/traffic-safety-nlp-dashboard.git

2. Install the required dependencies:
   pip install -r requirements.txt 

3. Run the Streamlit app:
   streamlit run traffic-safety-nlp-dashboard.py
